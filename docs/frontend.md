# Guia do Frontend

## Páginas
- Login
- Listagem de Clientes
- Cadastro/Edição de Cliente

## Componentes
- Formulário de Cliente
- Tabela de Clientes
- Inputs validados (CPF, Email)

## Acessibilidade e Responsividade
- Utiliza Tailwind CSS para garantir responsividade (mobile-first)
- Segue padrões WAI-ARIA para acessibilidade

## Integração
- Comunicação com API via hooks customizados (`useClientes`, `useAuth`)
- Configuração da URL da API via `.env.example`
